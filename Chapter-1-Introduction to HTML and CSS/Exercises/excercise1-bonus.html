HTML & CSS Quick Reference Guide:

HTML Tags:

Paragraph Tag (p)

The HTML paragraph element defines a paragraph

Header Tag (h)

The HTML header element defines a heading and inserts a title.

List Tag (li)

The HTML list element defines a list items and is used in ordered lists (ol) and unordered lists (ul).

Table Tag (table)

The HTML table element defines a table. Uses (tr) to insert rows and (td) to insert columns.

CSS Tags:

var()

Inserts the value of a custom property.

rgb ()

Defines colors using the Red-Green-Blue model.

max ()

Uses the largest value, from a comma-separated list of values, as the property value.

attr ()

Returns the value of an attribute of the selected element.
